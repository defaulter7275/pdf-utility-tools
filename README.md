# 🛠️ PDF Utility Tools

![Python](https://img.shields.io/badge/Python-3.10%2B-blue?logo=python&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-green)
![Maintenance](https://img.shields.io/badge/Maintained%3F-yes-brightgreen.svg)

A collection of lightweight and efficient Python scripts designed to automate common PDF workflows. Whether you need to merge certificates, insert images, or compress large documents, these tools provide a programmatic solution.

---

## 🚀 Key Features

* **📄 PDF Merger**: Automatically scans a directory, sorts PDF files, and combines them into a single high-quality document.
* **🖼️ Image-to-PDF Inserter**: Converts an image (like a certificate or cover) into a full-page PDF and inserts it at a specific page index.
* **📉 Hard Compression**: Utilizes advanced garbage collection and stream deflation to significantly reduce file sizes while maintaining integrity.

## 🛠️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/defaulter7275/pdf-utility-tools.git
cd pdf-utility-tools
```
### 2. Install dependencies
```bash
pip install -r requirements.txt
```
